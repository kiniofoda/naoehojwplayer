this is a Google Drive Free + Skins Video script

Original script by WG Tutoriales

how to install
1. Upload to your server & Extract / Unzip the package.

how to run
1. yourhosting.com/gplayer/?id=(id google drive file) or yourhosting.com/gplayer/?id=0B7O1IJJQh7NpalZQb2tnazBEcDA

1. https://drive.google.com/file/d/0B7O1IJJQh7NpalZQb2tnazBEcDA/edit ==> yourhosting.com/gplayer/?id=0B7O1IJJQh7NpalZQb2tnazBEcDA
